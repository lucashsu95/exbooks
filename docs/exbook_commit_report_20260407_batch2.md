## 功能與修改摘要

### 1. 圖片與封面邏輯規範化
針對書籍圖片的顯示機制做了全面重構：
- **Fallback 機制**：統一了全站圖片在缺失時的替代邏輯，確保在實拍照片功能上線後，能流暢切換官方封面與使用者上傳的照片。
- **資料清理**：清理了開發初期的假圖資料，並將「官方書目」與「實拍照片」的顯示層級定義得更清晰。

### 2. 關鍵 Bug 修復
修復了數個影響使用者操作的嚴重問題：
- **搜尋與過濾**：解決了所有書籍頁面中 URL 參數重複堆疊的問題，並修正了 `book_all` 頁面偶爾無法正確載入資料的 Bug。
- **權限與邏輯**：修復了書況照片的上傳權限檢查，並優化了 ISBN 查詢後的封面顯示邏輯。
- **交易管理**：改善了交易列表的顯示邏輯，現在會隱藏已評價的交易，並將待評價項目直接導向詳情頁。

### 3. 技術債重構與測試提升 (重點)
- **服務層拆分**：進行了大規模的技術債清理，將過於臃腫的 Logic 拆分到專門的服務層，提升程式碼的可讀性與維護性。
- **覆蓋率達 82%**：新增了多項測試案例，確保核心邏輯的穩定，目前的整體測試覆蓋率已顯著提升至 82%。
- **異常處理**：修正了信任等級計算中可能出現的除零錯誤，增強了系統的健壯性。

### 4. 前端合規與驗證強化
- **無障礙支援**：根據 Web Interface Guidelines 改善了全站的無障礙支援，讓平臺更具包容性。
- **表單驗證**：強制要求書籍上架時必須上傳實拍照片，並強化了對應的後端驗證。
- **違規統計**：新增了使用者違規次數的自動統計功能，為後續的停權機制提供準確數據。
